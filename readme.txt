Simple AJAX based Currency Converter Widget, data is fetched from Yahoo! Finance.

== Description ==

Adds a widget for simple currency conversion.

* Data is fetched from Yahoo! Finance web site per each request, making it up-to-date every time.
* AJAX feature provides smooth experience.

== Installation ==

1. Download the zip file and extract the contents,
2. Upload the folder simcurrex to your WP plugin folder /wp-content/plugins/ directory,
3. Go to Plugins > Installed, and activate the plugin,
4. Go to Appearance > Widgets, and drag the widget to a sidebar,
5. Select widget title and default currencies

== Frequently Asked Questions ==

= Where does the currency conversion data come from? =

From Yahoo! Finance web site - http://finance.yahoo.com/.

= How frequently are the values updated? =

Values are updated on each currency conversion, making it most up-to-date as possible.

= Any special PHP configuration? =

Your configuration has to allow access to remote servers.
